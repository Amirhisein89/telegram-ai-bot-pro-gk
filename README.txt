         راهنمای نصب و اجرای بات
═══════════════════════════════════════════════

مرحله ۱ — نصب کتابخانه‌ها
───────────────────────────
در ترمینال (CMD یا PowerShell) بنویس:

    pip install aiogram==3.13.1 aiohttp==3.10.10 aiosqlite==0.20.0


مرحله ۲ — تنظیم فایل config.py
────────────────────────────────
فایل config.py رو باز کن و این دو خط رو ویرایش کن:

    BOT_TOKEN = "توکن بات از BotFather"
    ADMIN_ID  = آی‌دی عددی خودت از @userinfobot


مرحله ۳ — اجرای بات
─────────────────────
در پوشه بات، در ترمینال بنویس:

    python bot.py


═══════════════════════════════════════════════
            دستورات ادمین
═══════════════════════════════════════════════

/addcode کد          ← اضافه کردن کد اشتراک جدید
/delcode کد          ← حذف کد
/codes               ← لیست همه کدها و وضعیتشون
/users               ← لیست کاربران
/help                ← راهنمای ادمین


═══════════════════════════════════════════════
            دستورات کاربران
═══════════════════════════════════════════════

/start               ← شروع
/activate کد         ← فعال‌سازی اشتراک
/status              ← وضعیت اشتراک
تصویر: توضیح         ← ساخت تصویر با هوش مصنوعی
هر سوال دیگه‌ای      ← پاسخ از هوش مصنوعی


═══════════════════════════════════════════════
            نکات مهم
═══════════════════════════════════════════════

- دیتابیس به صورت خودکار ساخته می‌شه (bot_database.db)
- هر کد اشتراک فقط یک بار قابل استفاده‌ست
- مدت اشتراک ۷ روز از زمان فعال‌سازیه
- همه پیام‌ها، عکس‌ها و استارت‌ها به ادمین اطلاع داده می‌شه
- عکس‌های ارسالی کاربران مستقیم برای ادمین فوروارد می‌شه
